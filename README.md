MÔ TẢ WEBSITE CÁ NHÂN
📌 1. Chủ đề

Website cá nhân (Portfolio) giới thiệu bản thân của Trương Anh Phú, tập trung vào:

Lập trình web (Frontend)
Kỹ năng và quá trình học tập
Các dự án đã thực hiện
📄 2. Nội dung chính

Website bao gồm các nội dung:

👤 Giới thiệu (About):
Thông tin cá nhân, mục tiêu, định hướng học tập
💪 Kỹ năng (Skills):
HTML, CSS, Responsive Design (hiển thị dạng thanh %)
📅 Hành trình (Timeline):
Quá trình học từ năm 2024 → 2026
🖼️ Dự án (Projects):
Các website đã làm (có hình ảnh minh họa)
🔐 Liên hệ (Contact):
Đăng nhập giả lập → sau đó gửi thông tin liên hệ
📑 3. Số trang
Website gồm 1 trang duy nhất (Single Page - SPA)
Điều hướng bằng menu → cuộn mượt đến từng phần
🎨 4. Giao diện (UI)
🔝 Header
Thanh menu ngang (About, Skills, Timeline, Projects, Contact)
Có highlight khi cuộn
🌟 Hero (Trang đầu)
Ảnh đại diện tròn
Tên + mô tả ngắn
Background gradient hiện đại
👤 About
Dạng card bo góc
Nội dung rõ ràng, dễ đọc
💪 Skills
Thanh kỹ năng dạng progress bar
Màu sắc đơn giản, trực quan
📅 Timeline
Dạng timeline dọc (giống CV)
Hiển thị theo năm
🖼️ Projects
Dạng grid (lưới)
Có hình ảnh + tiêu đề
Hover nổi lên
🔐 Contact
Gồm 2 phần:
Form đăng nhập
Form liên hệ (hiện sau khi login)
Có hiệu ứng báo lỗi + phản hồi
✨ 5. Hiệu ứng & trải nghiệm
Smooth scroll (cuộn mượt)
Animation xuất hiện khi cuộn
Hover card nâng lên
Button ripple effect
Responsive (mobile + desktop)
🎯 6. Mục đích
Giới thiệu bản thân
Trình bày kỹ năng và dự án
Làm portfolio cá nhân
Làm nền tảng học lập trình web
